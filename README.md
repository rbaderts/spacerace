Build & Run 
-----------

A multi-user asteroids clone.

      * golang backend
      * PIXI frontend
      * homegrown serverside physics implementation
      * network physics snapshot interpolation
      * Flatbuffers serialization
      * postgres
      
                    
* Install and setup latest golang 


`% ./build.sh` <br>
`% ./spacerace server` <br>
`localhost:8080` <br>
